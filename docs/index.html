YouTube スマートタブ - YouTube Smart Tabs （ツール）を有効にする

ツールを有効化する方法は2通りあります．YouTubeを開いている状態でツールが有効または無効に変更されたときは自動でリロードされます．

  • ブラウザの右上にある拡張機能のアイコンをクリック

    この拡張機能のアイコンが表示されていれば有効，アイコンの色がグレーであれば無効です．

または

  • 当サイト の上部にある「YouTube スマートタブ - YouTube Smart Tabs を有効にする」のトグルボタンをクリックして有効化

YouTubeの動画ページを開く

  1. ツールが有効になった状態でYouTubeを開き，視聴する動画ページを開きます．

    ※この拡張機能はYouTubeのトップページではなく，動画ページ（shortsを除く）のみ対応しています．

    動画ページのURLは，https://www.youtube.com/watch?v=VIDEO_ID

設定を変更する

  1. 動画ページを開くと，この拡張機能によりサイドバー（セカンダリエリア）に タブ と 選択されたタブのコンテンツ が表示されます．
  2. 設定タブをクリックすると，設定のメニューが表示されます．
  3. 設定メニューから以下の項目を設定できます．
    • タブ化するコンテンツ
    • 最初に表示するコンテンツ

    タブ化するコンテンツはデフォルトではコメント欄，関連動画，再生リストがタブ化されています．

    最初に表示するコンテンツはデフォルトで「自動（推奨）」の設定により，先頭のコンテンツまたは前回に選択していたタブ（設定タブ以外）が動画ページに移動した際に表示されます．

このドキュメントでは，YouTubeの動画ページに関連する各エリアや機能を一貫して説明するために，いくつかの用語を定義しています． 一般的な用語とは異なる意味や範囲で使用している場合もあるため，以下の定義を前提としてご参照ください．
  • ツール：tool
    本拡張機能が提供する，YouTubeの動画ページ上で動作する機能群を指します．ツールは，特定のエリア（例：コメント，チャット，概要など）に対して，表示の最適化や操作性の向上を目的として動作します．
  • タブ：tab
    本拡張機能によってYouTubeの動画ページに表示されるコンテンツの種類ごとに切り替え可能なナビゲーション要素．「概要」「チャット」「関連動画」「コメント」などの各エリアは，それぞれのタブによって表示が切り替わる．
  • コンテンツ：content
    YouTubeの動画ページにおいて表示される動画本体やその周辺情報（概要，コメント，チャット，関連など）を含む全体的な情報のことを指す．このドキュメントでは，タブごとに切り替えられる各エリアをまとめて「コンテンツ」と呼ぶ．
  • サイドバー：secondary
    画面の横（右側）に表示されるサブエリア（セカンダリエリアとも呼ばれる）で，通常，メインコンテンツの隣に位置する．拡張機能では，コメント欄をこの位置に表示できる．
  • 設定：setting
    この拡張機能を管理するための設定画面を指す．
  • 概要：description
    本来は動画の「概要欄（再生回数・投稿日）」を指すが，ここでは概要欄に加えて，動画のタイトル，チャンネル名，評価などの情報を含む動画メタデータ全体のエリアを指す．
  • チャット：chat
    YouTubeの動画ページにおいて，ライブ配信中またはライブ配信終了後のリプレイ動画に表示されるチャットのエリアを指す．
  • コメント：comments
    YouTubeの動画ページにおいて，視聴者が動画に対する意見や感想を投稿できるエリア．ライブ配信中のチャットと混同されがちだが，コメントは動画公開後も残り続け，時系列で並び替えられる．
  • 関連：related
    YouTubeの動画ページにおいて，現在表示されている動画に関連する他の動画が一覧で表示されるエリア．
  • 再生リスト：playlist
    YouTubeで複数の動画をまとめて整理・再生できる機能．動画ページでは，再生リストに含まれている場合，タブに表示される．
  • 動画プレイヤー：player
    動画の再生を担当するインターフェースやコンポーネント．YouTubeのページでは，動画を再生するためのボタン，音量調整，進行状況バー，再生停止ボタンなどが含まれたUIを指す．
  • 動画メタデータ：metadata
    動画に関連する情報や属性のこと．YouTubeでは，タイトル，チャネル名，評価（高評価・低評価），概要欄，公開日，再生回数などが動画メタデータに含まれる．
  • Large レイアウト：
    ウィンドウの横幅が1017px以上の状態．サイドバーが表示される．
  • Medium レイアウト：
    ウィンドウの横幅が1017px未満の状態．サイドバーが表示されない．

当サイトは YouTube Smart Tabs に関するマニュアルおよびバージョン情報などをまとめた拡張機能の専用サイトです．
拡張機能のアイコンまたは，任意のWEBサイト上で右クリックして拡張機能のメニューから当サイトを開くことができます．

説明:

YouTube Smart Tabs は，YouTube の 動画ページのUI を一新し，コメント欄や動画説明，関連動画などをタブ化することで動画視聴をより快適にします．複雑なページをスクロールする必要はもうありません．必要な情報をタブ化し，動画を観ながらコメント欄や概要を簡単に閲覧できます．

YouTubeの動画を視聴しながら，コメントや概要を手軽に確認できるので，「動画を観ながらコメントや概要をチェックしたい」という方におすすめの拡張機能です．

主な機能:

  • タブ型インターフェース： コメント，動画情報，関連動画などのコンテンツを個別のタブに分けて整理．
  • レイアウトのカスタマイズ： タブ化するコンテンツを自由に設定．
  • 視聴しながら操作可能： 動画を再生しながら，他のタブを簡単に操作．
  • ダークモード対応： ライトモードでもダークモードでも快適に使用可能．

詳細仕様:

  • サイドバー表示： ウィンドウ幅が 1017px 以上の場合，サイドバー（セカンダリエリア）にタブを表示．
  • メインエリア表示： ウィンドウ幅が 1017px 以下の場合，タブをメインエリア（プライマリエリア）に表示．
  • タブ化する情報： コメント欄，概要欄，関連動画などをタブ化．設定で表示する情報を変更可能．

注意点:

  • 一部の実験的なYouTubeレイアウトや機能とは互換性がない場合があります．
  • YouTubeの仕様変更により，拡張機能の動作が一時的に不安定になることがあります．
  • 他の拡張機能（特にUIを変更するタイプのもの）と併用すると，競合が発生する可能性があります．
